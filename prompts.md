# 不知道写什么

觉得今天没东西写的时候，看看下面的问题。不用全答，挑一个有感觉的写一句就好。

<div class="prompt-group">

### 事件

<ul class="prompt-list">
<li>今天发生了什么小事？</li>
<li>今天最占时间的事是什么？</li>
<li>今天有没有一个瞬间让我停了一下？</li>
</ul>
</div>

<div class="prompt-group">

### 情绪

<ul class="prompt-list">
<li>今天最明显的情绪是什么？</li>
<li>今天什么时候情绪波动最大？</li>
<li>今天我有没有突然低落、焦虑、烦躁？</li>
</ul>
</div>

<div class="prompt-group">

### 逃避

<ul class="prompt-list">
<li>今天我逃避了什么？</li>
<li>我今天最不想面对什么？</li>
<li>我是不是把某件小事想得太大了？</li>
</ul>
</div>

<div class="prompt-group">

### 身体

<ul class="prompt-list">
<li>昨晚睡得怎么样？</li>
<li>今天身体最明显的感觉是什么？</li>
<li>我有没有把身体疲惫误解成自己能力差？</li>
</ul>
</div>

<div class="prompt-group">

### 自省

<ul class="prompt-list">
<li>今天暴露了什么旧模式？</li>
<li>我今天对自己有什么误解？</li>
<li>今天有什么事情反复出现？</li>
</ul>
</div>

<div class="prompt-group">

### 一善

<ul class="prompt-list">
<li>今天做得还可以的一点是什么？</li>
<li>今天有没有一个很小但没有放弃的动作？</li>
<li>今天我有没有比以前稍微好一点点？</li>
</ul>
</div>

<div class="prompt-group">

### 自警

<ul class="prompt-list">
<li>今天最想留给明天的一句话是什么？</li>
<li>如果只写一句提醒自己，会是什么？</li>
<li>下次遇到类似问题，我该提醒自己什么？</li>
</ul>
</div>
