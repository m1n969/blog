# 关于此站

这是一个公开的日课与反省录。

不是技术博客，不是心理咨询档案。只是每天写一点，观察自己，记录模式，慢慢改。

## 日课规则

- 一句也算一日。
- 不补旧账，只接今日。
- 反省不是自责。
- 记录事实，观察模式。
- 不求漂亮，只求不断。
- 有事记事，无事记心。
- 强烈情绪可以先记录，不急着解释。
- 今天写得短，不代表今天失败。

## 最低完成标准

只要写完**睡眠记录 + 今日一句**，就算今天不断线。

> 有事记事，无事记心，心也无波，记身体，身体也无感，记一句。

## 如何炼一句自警

自警语不是鸡汤，也不是硬写名言。它来自当天的具体问题。

步骤：

1. 先写今天的具体卡点。
2. 去掉细节，找出重复模式。
3. 用一句短话提醒自己下次怎么做。
4. 不写空泛鼓励。
5. 不写自我羞辱。

**例子：**

具体问题：今天因为害怕 GitHub 太复杂，所以没有开始。

重复模式：把任务想得过大，导致逃避。

自警语：**畏难之时，只取第一步。**

## 自警语四类

- **戒**：提醒自己不要做什么
- **勉**：提醒自己做什么
- **识**：记录一个认识
- **规**：给自己定一条小规则

## 关于断更

断了不用补。不用解释为什么断了。直接写今天的就好。

> 断了也可以继续，不必补罪。
